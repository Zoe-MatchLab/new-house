## 三、指标定义

### A. 内容发布指标

| 指标 | 说明 | 数据维度 |
|-----|------|---------|
| 发布量 | 各平台内容发布数量 | 小红书、公众号、视频号、朋友圈 |
| 浏览量 | 各平台内容曝光量 | 小红书、公众号、视频号、朋友圈 |
| 投流费用 | 付费推广消耗金额 | 各平台 |

**发布量明细**：

| 平台 | 指标 |
|-----|------|
| 小红书 | 发布量 |
| 公众号 | 发布量 |
| 视频号 | 发布量 |
| 朋友圈 | 发布量 |
| **合计** | 总发布量 |

**浏览量明细**：

| 平台 | 指标 |
|-----|------|
| 小红书 | 浏览量 |
| 公众号 | 浏览量 |
| 视频号 | 浏览量 |
| 朋友圈 | 浏览量 |
| **合计** | 总浏览量 |

**投流费用明细**：

| 平台 | 指标 |
|-----|------|
| 小红书 | 投流费用 |
| 公众号 | 投流费用 |
| 视频号 | 投流费用 |
| **合计** | 总投流费用 |

---

### B. 获客转化指标

| 指标 | 说明 |
|-----|------|
| 咨询量 | 客户主动咨询总次数 |
| 线索量 | 留资客户数量 |
| 报备量 | 客户报备楼盘数量 |
| 带看量 | 经纪人带看次数 |
| 获客成本 | 单个有效客户的获取成本 |

**核心漏斗**：

```
咨询量 → 线索量 → 报备量 → 带看量 → 成交
```

| 转化节点 | 指标 |
|---------|-----|
| 咨询→线索 | 留资率 |
| 线索→报备 | 报备率 |
| 报备→带看 | 带看率 |
| 带看→成交 | 成交率 |

---

### C. 产品力训练指标

| 指标 | 说明 |
|-----|------|
| 专家占比 | 各级别专家人数占比（S1/S2/S3/S4） |
| 平均能力分 | 团队整体能力均值 |
| 能力提升率 | 较上一周期能力提升幅度 |

**专家等级定义**：

| 等级 | 名称 | 要求 |
|-----|------|-----|
| S4 | 全城专家 | 全城90%以上楼盘得分≥80分 |
| S3 | 行政区专家 | 所在行政区80%以上楼盘得分≥80分 |
| S2 | 商圈专家 | 所在商圈80%以上楼盘得分≥80分 |
| S1 | 单盘专家 | 负责楼盘得分≥90分 |

## 五、小程序看板

### 5.1 经纪人个人看板

| 模块 | 说明 |
|-----|------|
| 作业概览 | 探盘任务数据指标 |
| 能力评分 | 当前能力等级、综合评分 |
| 任务进度 | 进行中任务、待提交任务 |

**能力评分展示**：
- 当前等级（S1/S2/S3/S4）
- 综合评分（0-100分）
- 各维度得分

**任务进度展示**：
- 进行中任务数量
- 待提交任务数量
- 逾期任务提醒

---

### 5.2 管理层看板（小程序端）

| 模块 | 说明 |
|-----|------|
| 团队业绩 | 团队整体数据、同比环比 |
| 排名展示 | 经纪人业绩排名 |
| 任务跟进 | 各任务执行进度、逾期提醒 |

**团队业绩**：
- 团队总业绩（GMV）
- 团队总单量
- 同比/环比展示

**排名展示**：
- 按业绩排名
- 按带看量排名
- 按报备量排名

**任务跟进**：
- 各任务执行进度
- 逾期提醒
- 未完成经纪人列表

## 六、核心业务漏斗

```
报备 → 带看 → 成交
```

| 转化节点 | 指标 | 公式 |
|---------|-----|-----|
| 报备→带看 | 报备转带看率 | 带看组数/报备组数 |
| 带看→成交 | 转化率 | 成交单量/带看组数 |
| 复访 | 复访率 | 同盘复访组数/带看组数 |
