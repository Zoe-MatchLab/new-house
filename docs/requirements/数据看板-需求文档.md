## 2. 业务流程

### 2.1 主流程

```
用户登录 → 选择看板类型 → 查看数据 → 筛选/刷新 → 导出/分享
```

### 2.2 数据刷新流程

```
用户点击刷新 → 请求后端数据 → 合并API+导入数据 → 展示
```

### 2.3 整体结构

```
新房数据看板（管理层）
├── 运营数据
│   ├── 重点楼盘（TOP3榜单）
│   ├── 价格动态
│   └── 案例展示
├── AI话术训练数据
│   ├── 日常训练（使用率+模块分析）
│   ├── 通关演练（通过率+分数）
│   └── 产品力成长体系（五维雷达图）
└── 业务数据
    ├── 报备数据
    ├── 带看数据（含三星考核）
    └── 业绩数据
```

## 3. 业务规则

### 3.1 数据权限规则

| 规则 | 说明 |
|-----|------|
| 查看权限 | 管理员可查看全部数据 |
| 刷新权限 | 用户主动刷新 |
| 数据隔离 | 按组织架构层级隔离 |

### 3.2 数据更新规则

| 规则 | 说明 |
|-----|------|
| 更新方式 | 用户主动刷新 |
| 公众号数据 | API实时获取 |
| 其他平台 | 数据导入后更新 |

### 3.3 数据统计规则

| 规则 | 说明 |
|-----|------|
| 时间统计 | 按自然月/自然周统计 |
| 门店统计 | 按归属门店统计 |
| 经纪人统计 | 按经纪人ID去重 |

## 4. 数据指标

### 4.1 核心指标

#### 4.1.1 运营数据

| 指标 | 定义 | 公式 | 优先级 |
|-----|------|-----|-------|
| 本月带看楼盘TOP3 | 带看量排名前3 | - | P0 |
| 本月认购楼盘TOP3 | 认购量排名前3 | - | P0 |
| 本月带看→成交转化比TOP3 | 转化效率排名 | 成交/带看 | P0 |
| 本区域报名楼盘TOP3 | 本区域报名量排名 | - | P1 |
| 特价房楼盘 | 当前特价房楼盘 | - | P1 |

#### 4.1.2 AI话术训练数据

| 指标 | 定义 | 公式 | 优先级 |
|-----|------|-----|-------|
| 整体使用率 | 参与训练占比 | 参与人数/总人数 | P0 |
| 高频模块 | 使用频率最高 | - | P1 |
| 整体通过率 | 通关通过比例 | 通过人数/参与人数 | P0 |
| 能力评分 | 综合能力分 | AI评分 | P0 |
| 掌握度 | 知识掌握程度 | 百分比 | P0 |

**五维能力维度**：
| 维度 | 说明 |
|-----|------|
| 产品力 | 楼盘产品知识 |
| 抗压能力 | 压力应对能力 |
| 沟通技巧 | 沟通表达能力 |
| 业务熟悉度 | 业务知识掌握 |
| 销冠经验值 | 销冠经验积累 |

**能力等级**：
| 等级 | 说明 |
|-----|------|
| 销冠 | 最高等级 |
| 高 | 较高等级 |
| 中 | 中等级 |
| 低 | 基础等级 |

#### 4.1.3 业务数据

| 指标 | 定义 | 公式 | 优先级 |
|-----|------|-----|-------|
| 有效报备次数 | 有效报备总次数 | - | P0 |
| 有效报备组数 | 有效报备组数 | - | P1 |
| 有效报备经纪人数 | 有报备的经纪人数量 | - | P1 |
| 报备活跃度 | 报备经纪人占比 | 有报备人数/总人数 | P1 |

| 三星带看占比 | 三星经纪人带看占比 | 三星带看组/总带看组 | P2 |
| 三星带看活跃度 | 三星经纪人占比 | 三星人数/总人数 | P2 |
| 总业绩（GMV） | 成交总金额 | - | P0 |
| 总单量 | 成交总单数 | - | P0 |
| 转化率 | 成交/带看比例 | 成交单量/带看组数 | P0 |
| 人均业绩 | 人均成交金额 | 总业绩/经纪人数 | P1 |

### 4.2 指标优先级汇总

| 优先级 | 指标 |
|-------|-----|
| P0 | 转化率、带看组数、总业绩GMV、整体通过率、能力评分、掌握度 |
| P1 | 报备活跃度、带看活跃度、人均业绩、高频模块 |
| P2 | 三星带看占比、三星带看活跃度 |

### 4.3 核心业务漏斗

```
报备 → 带看 → 成交
```

| 转化节点 | 指标 | 公式 |
|---------|-----|-----|
| 报备→带看 | 报备转带看率 | 带看组数/报备组数 |
| 带看→成交 | 转化率 | 成交单量/带看组数 |
| 复访 | 复访率 | 同盘复访组数/带看组数 |

## 6. 异常处理

| 异常情况 | 处理方式 |
|---------|---------|
| 数据加载失败 | 显示错误提示，提供重试按钮 |
| 部分数据缺失 | 展示已有数据，缺失数据显示"--" |
| API调用失败 | 降级为已导入数据展示 |
| 数据导入延迟 | 显示数据更新时间提示 |
