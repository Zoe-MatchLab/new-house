## 2. 业务流程

### 2.1 主流程

```
用户登录 → 选择看板类型 → 查看数据 → 筛选/刷新 → 导出/分享
```

### 2.2 数据刷新流程

```
用户点击刷新 → 请求后端数据 → 合并API+导入数据 → 展示
```

### 2.3 整体结构

```
新房数据看板（管理层）
├── 运营数据
│   ├── 重点楼盘（TOP3榜单）
│   └── 价格动态
├── AI话术训练数据
│   ├── 日常训练（使用率+模块分析）
│   ├── 通关演练（通过率+分数）
│   └── 产品力成长体系（五维雷达图）
└── 业务数据
    ├── 报备数据
    ├── 带看数据（含三星考核）
    └── 业绩数据
```

## 3. 业务规则

### 3.1 数据权限规则

| 规则 | 说明 |
|-----|------|
| 查看权限 | 管理员可查看全部数据 |
| 刷新权限 | 用户主动刷新 |
| 数据隔离 | 按组织架构层级隔离 |

### 3.2 数据更新规则

| 规则 | 说明 |
|-----|------|
| 更新方式 | 用户主动刷新 |
| 公众号数据 | API实时获取 |

### 3.3 数据统计规则

| 规则 | 说明 |
|-----|------|
| 时间统计 | 按自然月/自然周统计 |
| 门店统计 | 按归属门店统计 |
| 经纪人统计 | 按经纪人ID去重 |

### 4.2 AI话术训练数据

#### 4.2.1 日常训练

| 指标 | 说明 | 维度 | 状态 |
|-----|------|-----|------|
| 整体使用率 | 参与训练经纪人占比 | 全司/区域/门店 | ✅ |
| 高频模块分析 | 各训练模块使用频率排名 | 整体 | ✅ |

**使用率统计周期**：月度

**高频模块排名**：显示TOP3

**模块维度**（5大模块）：
1. 开场破冰&探需
2. 产品知识
3. 异议处理
4. 谈单逼定
5. 售后维护

**模块得分详情**：需要，但目前暂无数据（后续版本）

#### 4.2.2 通关演练

| 指标 | 说明 | 维度 | 状态 |
|-----|------|-----|------|
| 整体通过率 | 通关考试通过比例 | 全司/门店/区域/个人/楼盘 | ✅ |
| 本月通关分数 | 通关考试成绩 | 门店/区域/楼盘 | ✅ |

**通过率计算公式**：通过人数 / 参与人数

**历史趋势展示**：有

**通关分数**：取**平均分**

**交互要求**：
- 支持按门店/区域/个人/楼盘维度查看
- 显示历史趋势

#### 4.2.3 产品力成长体系

**页面结构**：
```
产品力成长体系
├── 顶部筛选：本周/本月/本季度/自定义
├── Tab切换：我的 | 团队
├── 核心指标卡片（4个）
├── 能力评估报告
│   ├── 五维雷达图
│   └── 各维度评分
├── 能力分布
│   └── 分布柱状图
└── 进步幅度
    └── 排行列表
```

**核心指标卡片**（4个）：

| 指标 | 说明 | 展示 | 状态 |

| 能力评分 | 综合能力分 | 数值+趋势（较上周） | ✅ |
| 掌握度 | 知识掌握程度 | 百分比+趋势 | ✅ |
| 训练时长 | 累计训练时长 | 数值+趋势 | ✅ |
| 训练人次 | 参与训练次数 | 数值+趋势 | ✅ |

**五维能力雷达图**：

| 维度 | 说明 |
|-----|------|
| 产品力 | 楼盘产品知识 |
| 抗压能力 | 压力应对能力 |
| 沟通技巧 | 沟通表达能力 |
| 业务熟悉度 | 业务知识掌握 |
| 销冠经验值 | 销冠经验积累 |

**能力分布**：

| 等级 | 说明 |
|-----|------|
| 销冠 | 最高等级 |
| 高 | 较高等级 |
| 中 | 中等级 |
| 低 | 基础等级 |

**进步幅度**：

| 指标 | 说明 |
|-----|------|
| 进步最大 | 进步最多的经纪人排名 |
| 进步分值 | 进步分数 |

**交互要求**：
- 支持时间筛选（本周/本月/本季度/自定义）
- 支持Tab切换（我的/团队）

- 点击经纪人可查看详情

### 4.3 业务数据

#### 4.3.1 报备数据

| 指标 | 说明 | 公式 | 状态 |
|-----|------|-----|------|
| 有效报备次数 | 有效报备总次数 | - | ✅ |
| 有效报备组数 | 有效报备组数 | - | ✅ |
| 有效报备经纪人数 | 有报备的经纪人数量 | - | ✅ |
| 报备活跃度 | 报备经纪人占比 | 有报备经纪人数 / 总经纪人数 | ✅ |

**有效报备定义**：被楼盘接受的报备

**交互要求**：
- 支持按时间筛选
- 支持按门店/区域查看
- 显示报备转化漏斗

#### 4.3.2 带看数据

**基础指标**

| 指标 | 说明 |
|-----|------|
| 带看次数 | 带看总次数 |
| 带看组数 | 带看总组数 |
| 同盘复访组数 | 同一楼盘多次带看 |

**活跃度**

| 指标 | 公式 |
|-----|------|
| 带看活跃度 | 带看经纪人数 / 总经纪人数 |

**三星考核**（经纪人评级体系）

| 指标 | 说明 | 公式 |
|-----|------|-----|
| 三星经纪人带看组数 | 三星经纪人完成的带看组数 | - |
| 三星带看占比 | 三星带看比例 | 三星带看组数 / 总带看组数 |
| 三星带看活跃度 | 三星经纪人占比 | 三星经纪人数 / 总经纪人数 |

**三星经纪人定义**：通过三星评定的经纪人

#### 4.3.3 业绩数据

| 指标 | 说明 | 公式 | 状态 |
|-----|------|-----|------|
| 总业绩（GMV） | 成交总金额（**万元**） | - | ✅ 已澄清 |
| 总单量 | 成交总单数 | - | ✅ |
| 转化率 | 成交/带看比例 | 成交单量 / 带看组数 | ✅ 已澄清 |
| 人均业绩 | 人均成交金额 | 总业绩 / 成交经纪人数 | ✅ 已澄清 |

**澄清说明**：
- GMV单位：**万元**
- 转化率公式：成交单量 / 带看组数
- 人均业绩分母：成交经纪人数

## 6. 异常处理

| 异常情况 | 处理方式 |
|---------|---------|
| 数据加载失败 | 显示错误提示，提供重试按钮 |
| 部分数据缺失 | 展示已有数据，缺失数据显示"--" |
| API调用失败 | 降级为已导入数据展示 |
| 数据导入延迟 | 显示数据更新时间提示 |
