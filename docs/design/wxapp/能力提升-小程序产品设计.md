## 1. 概述

### 1.1 设计目标

为经纪人提供便捷的话术训练和录音分析功能，随时随地提升能力。

### 1.2 页面清单

| 页面 | 功能 | 文件 |
|-----|------|-----|
| 训练首页 | 场景选择、开始训练 | training-index.html |
| AI对话 | 实时对话、即时反馈 | training-chat.html |
| 能力评估 | 综合评分、雷达图 | training-result.html |
| 录音分析 | 上传、分析、结果 | recording-analysis.html |
| 历史记录 | 训练记录、趋势图 | training-history.html |

## 5. 交互说明

### 5.1 训练流程

1. 选择楼盘 → 选择场景 → 开始训练
2. AI提问 → 经纪人回答 → AI即时反馈
3. 10轮后 → 显示综合评分 + 雷达图 + 建议
4. 可分享结果或再次训练

### 5.2 录音分析流程

1. 选择录音文件 → 上传（显示进度）
2. 进入分析页面 → 实时显示进度（上传→转写→分析→完成）
3. 分析完成 → 自动展示结果
4. 可关联客户或查看详情
