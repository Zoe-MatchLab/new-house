# 探盘任务 - 产品设计文档

> 版本：1.0
> 日期：2026-04-13
> 端：PC/Web端

---

## 1. 概述

### 1.1 设计目标

为区域经理提供探盘任务的配置、下发、追踪功能，确保经纪人执行标准化。

### 1.2 设计原则

- **高效下发**：减少配置步骤，快速下发任务
- **进度透明**：实时展示任务进度，异常及时预警
- **操作明确**：每个状态下对应明确的操作按钮

## 3. 页面详细设计

### 3.1 任务列表页

#### 页面结构

```
┌────────────────────────────────────────────────────────────────────┐
│  探盘任务                                              [+新建任务]    │
├────────────────────────────────────────────────────────────────────┤
│ [搜索楼盘] [状态筛选] [楼盘筛选] [重置] [搜索]                       │
├────────────────────────────────────────────────────────────────────┤
│ 楼盘名称 │ 关联任务       │ 状态      │ 完成进度   │ 截止时间 │ ... │
│──────────┼───────────────┼───────────┼────────────┼─────────┼────│
│ 龙湖·云景 │ [路书][训练]  │ [进行中]   │ ████░░ 18/30│ 04-20  │ ... │
│ 万科·翡翠 │ [路书][录音]  │ [待下发]   │ ░░░░░░  0/25│ 04-25  │ ... │
│ 中海·寰宇 │ [路书]        │ [已失效]   │ ██░░░░  9/30│ 04-10  │ ... │
├────────────────────────────────────────────────────────────────────┤
│                                          共 4 条记录     [< 1 >]    │
└────────────────────────────────────────────────────────────────────┘
```

#### 组件清单

| 组件 | ID | 用途 |
|-----|-----|-----|
| TaskCard | taskCard | 任务卡片（列表行） |
| TaskTags | taskTags | 任务类型标签组 |
| StatusBadge | statusBadge | 状态徽章 |
| ProgressBar | progressBar | 进度条 |
| ReminderConfig | reminderConfig | 提醒节点配置 |

#### 交互说明

- 搜索：实时搜索，防抖300ms
- 状态筛选：选择后自动刷新列表
- 新建任务：弹出配置弹窗
- 操作按钮：根据状态显示不同操作

### 3.2 新建任务弹窗

#### 页面结构

```
┌────────────────────────────────────────────┐
│  新建探盘任务                            [×] │
├────────────────────────────────────────────┤
│                                            │
│  选择楼盘 *    [请选择楼盘 ▼]               │
│                                            │
│  执行人 *      [请选择 ▼]                  │
│                                            │
│  可见范围      [全部可见 ▼]                 │
│                                            │
│  截止时间 *    [2026-04-20 18:00]         │
│                                            │
│  提醒节点                                 │
│  ┌────────────────────────────────────┐   │
│  │ 🟡 轻度提醒  [24] 小时前 [推送通知 ▼]  │   │
│  │ 🟠 中度提醒  [12] 小时前 [强提醒+短信▼]│   │
│  │ 🔴 严重提醒  [ 2] 小时前 [强提醒+电话▼] │   │
│  └────────────────────────────────────┘   │
│                                            │
├────────────────────────────────────────────┤
│                         [取消] [确认下发]   │
└────────────────────────────────────────────┘
```

#### 组件清单

| 组件 | ID | 用途 |
|-----|-----|-----|
| Select |楼盘选择 | 下拉选择楼盘 |
| Select |执行人选择 | 组织架构或指定人员 |
| DateTimePicker |截止时间 | 日期时间选择 |
| ReminderItem |提醒项 | 提醒时间+方式配置 |

#### 交互说明

- 楼盘选择：必选，联动显示楼盘信息
- 执行人：必选，支持按组织架构或指定人员
- 提醒节点：可配置多个，默认3个（轻度/中度/严重）
- 确认下发：校验必填后下发，关闭弹窗，刷新列表
