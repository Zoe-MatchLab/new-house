# 能力提升 - 产品设计文档

> 版本：1.0
> 日期：2026-04-13
> 端：PC/Web端

---

## 1. 概述

### 1.1 设计目标

为运营人员提供题库配置功能，为区域经理提供团队训练数据分析。

### 1.2 页面清单

| 页面 | 功能 | 文件 |
|-----|------|-----|
| 题库管理 | 配置各楼盘训练题库 | question-bank.html |
| 训练数据 | 查看经纪人训练记录 | training-data.html |
| 能力看板 | 团队/个人能力雷达图 | ability-dashboard.html |

## 5. 原型说明

### 5.1 原型入口

文件：`../prototypes/pc/能力提升-产品设计.html`

### 5.2 交互说明

| 交互 | 说明 |
|-----|-----|
| 添加题库 | 弹出Modal，选择楼盘和场景，编辑题目 |
| 编辑题目 | 进入编辑页面，保存后刷新列表 |
| 查看雷达图 | 点击成员名称，展开详细雷达图 |
| 导出数据 | 选择格式（Excel/PDF），下载文件 |
