# 数据看板 - 完整需求文档

> 版本：1.0
> 日期：2026-04-13
> 状态：需求已澄清
> 端：PC端管理后台

---

## 一、产品定位

### 1.1 目标用户
- 管理层（区域经理及以上）

### 1.2 核心价值
- 实时掌握全司业务数据
- 监控核心漏斗转化
- 发现问题并决策

### 1.3 数据权限
- 管理员权限，可查看全部数据
- 用户主动刷新

---

## 二、整体结构

```
新房数据看板（管理层）
├── 运营数据
│   ├── 重点楼盘
│   ├── 价格动态
│   └── 案例展示
├── AI话术训练数据
│   ├── 日常训练
│   ├── 通关演练
│   └── 产品力成长体系 【新增】
└── 业务数据
    ├── 报备数据
    ├── 带看数据
    └── 业绩数据
```

---

## 三、核心业务漏斗

```
报备 ────→ 带看 ────→ 成交
          ↑
       （复访）
```

| 转化节点 | 指标 |
|---------|-----|
| 报备→带看 | 报备转带看率 |
| 带看→成交 | 带看转成交率（核心指标） |
| 复访 | 同盘复访率 |

## 四、功能模块详情

### 4.1 运营数据

#### 4.1.1 重点楼盘

| 指标 | 说明 | 数据维度 |
|-----|------|---------|
| 本月带看楼盘TOP3 | 全司带看量排名前3 | 全司 |
| 本月认购楼盘TOP3 | 全司认购量排名前3 | 全司 |
| 本月带看→成交转化比TOP3 | 转化效率排名前3 | 全司 |
| 本区域报名楼盘TOP3 | 本区域报名量排名前3 | 本区域 |
| 特价房专区 | 当前特价房楼盘列表 | 全司 |

**交互要求**：
- 点击楼盘名称可跳转查看详情
- 支持按区域/门店筛选

#### 4.1.2 价格动态

| 指标 | 说明 | 更新频率 |
|-----|------|---------|
| 本月调价楼盘 | 本月价格变动的楼盘列表 | 月度 |
| 本月聚焦楼盘均价 | 重点楼盘当前均价 | 月度 |

**交互要求**：
- 显示价格变动幅度（↑↓）
- 支持按时间筛选

#### 4.1.3 案例展示

| 指标 | 说明 |
|-----|------|
| 本月成交典型案例 | 成交案例分析（含客户画像、成交过程） |

**交互要求**：
- 点击展开案例详情
- 支持案例库检索

### 4.2 AI话术训练数据

#### 4.2.1 日常训练

| 指标 | 说明 | 维度 |
|-----|------|-----|
| 整体使用率 | 参与训练经纪人占比 | 全司/区域/门店 |
| 高频模块分析 | 各训练模块使用频率排名 | 整体 |

**模块维度**（5大模块）：
1. 开场破冰&探需
2. 产品知识
3. 异议处理
4. 谈单逼定
5. 售后维护

**能力输出**：
- 团队能力总结（雷达图）
- 各模块得分分布
- 优化建议（AI生成）

#### 4.2.2 通关演练

| 指标 | 说明 | 维度 |
|-----|------|-----|
| 整体通过率 | 通关考试通过比例 | 全司/门店/区域/个人/楼盘 |
| 本月通关分数 | 通关考试成绩 | 门店/区域/楼盘 |

**交互要求**：
- 支持按门店/区域/个人/楼盘维度查看
- 显示历史趋势

#### 4.2.3 产品力成长体系 【新增】

> 参考原型：产品力成长体系数据看板

**页面结构**：

```
产品力成长体系
├── 顶部筛选：本周/本月/本季度/自定义
├── Tab切换：我的 | 团队
├── 核心指标卡片（4个）
├── 能力评估报告
│   ├── 五维雷达图
│   └── 各维度评分
├── 能力分布
│   └── 分布柱状图
└── 进步幅度
    └── 排行列表
```

**核心指标卡片**（4个）：

| 指标 | 说明 | 展示 |
|-----|------|-----|
| 能力评分 | 综合能力分 | 数值+趋势（较上周） |
| 掌握度 | 知识掌握程度 | 百分比+趋势 |
| 训练时长 | 累计训练时长 | 数值+趋势 |
| 训练人次 | 参与训练次数 | 数值+趋势 |

**五维能力雷达图**：

| 维度 | 说明 |
|-----|------|
| 产品力 | 楼盘产品知识 |
| 抗压能力 | 压力应对能力 |
| 沟通技巧 | 沟通表达能力 |
| 业务熟悉度 | 业务知识掌握 |
| 销冠经验值 | 销冠经验积累 |

**能力分布**：

| 等级 | 说明 |
|-----|------|
| 销冠 | 最高等级 |
| 高 | 较高等级 |
| 中 | 中等级 |
| 低 | 基础等级 |

**进步幅度**：

| 指标 | 说明 |
|-----|------|
| 进步最大 | 进步最多的经纪人排名 |
| 进步分值 | 进步分数 |

**交互要求**：
- 支持时间筛选（本周/本月/本季度/自定义）
- 支持Tab切换（我的/团队）
- 雷达图支持hover查看各维度详情
- 点击经纪人可查看详情

### 4.3 业务数据

#### 4.3.1 报备数据

| 指标 | 说明 | 公式 |
|-----|------|-----|
| 有效报备次数 | 有效报备总次数 | - |
| 有效报备组数 | 有效报备组数 | - |
| 有效报备经纪人数 | 有报备的经纪人数量 | - |
| 报备活跃度 | 报备经纪人占比 | 有报备经纪人数 / 总经纪人数 |

**交互要求**：
- 支持按时间筛选
- 支持按门店/区域查看
- 显示报备转化漏斗

#### 4.3.2 带看数据

**基础指标**

| 指标 | 说明 |
|-----|------|
| 带看次数 | 带看总次数 |
| 带看组数 | 带看总组数 |
| 同盘复访组数 | 同一楼盘多次带看 |

**活跃度**

| 指标 | 公式 |
|-----|------|
| 带看活跃度 | 带看经纪人数 / 总经纪人数 |

**三星考核**（经纪人评级体系）

| 指标 | 说明 | 公式 |
|-----|------|-----|
| 三星经纪人带看组数 | 三星经纪人完成的带看组数 | - |
| 三星带看占比 | 三星带看比例 | 三星带看组数 / 总带看组数 |
| 三星带看活跃度 | 三星经纪人占比 | 三星经纪人数 / 总经纪人数 |

**复访率**

| 指标 | 公式 |
|-----|------|
| 复访率 | 同盘复访组数 / 带看组数 |

**交互要求**：
- 支持按门店/区域/个人筛选
- 显示三星经纪人分布

#### 4.3.3 业绩数据

| 指标 | 说明 | 公式 |
|-----|------|-----|
| 总业绩（GMV） | 成交总金额 | - |
| 总单量 | 成交总单数 | - |
| 转化率 | 成交/带看比例 | 成交单量 / 带看组数 |
| 人均业绩 | 人均成交金额 | 总业绩 / 经纪人数 |

**交互要求**：
- 支持按门店/区域/个人筛选
- 显示同比/环比趋势

## 七、原型设计说明

### 7.1 页面结构

```
数据看板首页
├── 顶部统计卡片（4个核心指标）
├── 核心漏斗图（报备→带看→成交）
├── Tab切换
│   ├── 运营数据
│   │   ├── 重点楼盘（TOP3榜单）
│   │   ├── 价格动态
│   │   └── 案例展示
│   ├── AI话术训练
│   │   ├── 日常训练（使用率+模块分析）
│   │   └── 通关演练（通过率+分数）
│   └── 业务数据
│       ├── 报备数据
│       ├── 带看数据（含三星考核）
│       └── 业绩数据
└── 筛选条件（时间/区域/门店）
```

### 7.2 组件清单

| 组件 | 说明 |
|-----|-----|
| StatCard | 统计卡片（数值+趋势） |
| FunnelChart | 漏斗图（转化可视化） |
| RankingList | 排行榜（TOP3） |
| TabPanel | Tab切换面板 |
| RadarChart | 雷达图（能力分析） |
| TrendChart | 趋势图（同环比） |
| DataTable | 数据表格（明细） |
| FilterBar | 筛选条件栏 |
