> 版本：1.0
> 日期：2026-04-13
> 状态：需求已澄清

---

## 一、产品定位

### 1.1 目标用户
- 管理层（区域经理及以上）
- 经纪人（小程序端个人看板）

### 1.2 端分布

| 端 | 定位 | 使用者 |
|---|-----|-------|
| PC管理后台 | 管理层视角数据看板 | 区域经理及以上 |
| 小程序看板 | 经纪人个人+管理层看板 | 经纪人、管理层 |

---

## 二、整体结构

```
新房数据看板
├── PC管理后台看板
│   ├── A. 内容发布指标
│   ├── B. 获客转化指标
│   ├── C. 产品力训练指标
│   └── 数据导出（Excel/PDF）
└── 小程序看板
    ├── 经纪人个人看板
    │   ├── 作业概览
    │   ├── 能力评分
    │   └── 任务进度
    └── 管理层看板
        ├── 团队业绩
        ├── 排名展示
        └── 任务跟进
```

---

### 3.2 B. 获客转化指标

| 指标 | 说明 |
|-----|------|
| 咨询量 | 客户主动咨询总次数 |
| 线索量 | 留资客户数量 |
| 报备量 | 客户报备楼盘数量 |
| 带看量 | 经纪人带看次数 |
| 获客成本 | 单个有效客户的获取成本 |

**核心漏斗**：

```
咨询量 → 线索量 → 报备量 → 带看量 → 成交
```

---

### 3.3 C. 产品力训练指标

| 指标 | 说明 |
|-----|------|
| 专家占比 | 各级别专家人数占比 |
| 平均能力分 | 团队整体能力均值 |
| 能力提升率 | 较上一周期能力提升幅度 |

**专家等级定义**：

| 等级 | 名称 | 要求 |
|-----|------|-----|
| S4 | 全城专家 | 全城90%以上楼盘得分≥80分 |
| S3 | 行政区专家 | 所在行政区80%以上楼盘得分≥80分 |
| S2 | 商圈专家 | 所在商圈80%以上楼盘得分≥80分 |
| S1 | 单盘专家 | 负责楼盘得分≥90分 |

---

### 3.4 数据导出

| 格式 | 说明 |
|-----|------|
| Excel | 支持数据明细导出 |
| PDF | 支持报表导出 |
