## 1. 背景与目标

### 1.1 需求背景

**为什么要做这个需求？**

- 经纪人需要快速掌握楼盘产品力和销售话术
- 传统培训依赖人工考核，效率低且反馈慢
- 需要AI辅助练习，随时随地提升能力

**话术训练的价值**：

| 价值 | 说明 |
|-----|------|
| 随时练习 | 不受时间地点限制 |
| 即时反馈 | AI实时评分和建议 |
| 个性化 | 根据楼盘和场景定制 |

### 1.2 目标

**做完之后期望达到什么效果？**

- 经纪人可选择楼盘和场景进行AI对话练习
- 快速掌握产品卖点
- 获得AI即时反馈，知道改进方向

## 5. 核心业务流程

### 5.1 话术训练流程

```
选择楼盘
    ↓
选择场景（产品介绍/异议处理/逼单等）
    ↓
开始训练
    ↓
AI扮演客户提问
    ↓
经纪人回答
    ↓
AI评分 + 改进建议（每轮）
    ↓
继续下一轮（10-15轮）
    ↓
训练完成，生成报告
```

### 5.2 AI对话机制

| 机制 | 说明 |
|-----|------|
| AI角色 | 扮演潜在客户 |
| 提问类型 | 根据场景自动生成 |
| 评分标准 | 话术专业度、应变能力、转化意识 |
| 反馈时机 | 每轮回答后即时反馈 |

## 6. 功能详情

### 6.1 场景配置

| 配置项 | 说明 |
|-----|------|
| 楼盘 | 关联的新盘 |
| 场景类型 | 产品介绍/异议处理/约看/逼单/全流程 |
| 题库 | 该场景下的问题列表 |

### 6.2 AI评分维度

| 维度 | 说明 | 权重 |
|-----|------|------|
| 话术专业度 | 是否使用标准话术 | 30% |
| 产品熟悉度 | 对楼盘信息掌握程度 | 25% |
| 应变能力 | 处理异议的能力 | 25% |
| 转化意识 | 约看/逼单意识 | 20% |

### 6.3 训练报告

| 内容 | 说明 |
|-----|------|
| 总评分 | 综合评分（1-100分） |
| 各维度评分 | 各维度得分 |
| AI建议 | 具体改进建议 |
| 优秀回答 | 回答中表现好的部分 |
