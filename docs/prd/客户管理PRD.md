## 1. 背景与目标

### 1.1 需求背景

**为什么要做这个需求？**

- 经纪人在小程序侧获客时需要记录客户信息
- 管理层需要查看和统计客户数据
- 客户信息需要在PC端和小程序端同步

**当前存在什么问题？**

- 客户信息分散在各个渠道，无统一管理
- 经纪人无法便捷地维护客户信息
- 管理层无法查看客户汇总数据

**不做会怎样？**

- 客户跟进效率低
- 数据无法统一分析
- 客户资源流失

### 1.2 目标

**做完之后期望达到什么效果？**

- 经纪人可在小程序侧快速录入和维护客户信息
- 管理层可在PC端查看客户汇总数据
- 客户信息双向同步

**如何衡量成功？**

| 指标 | 目标值 |
|-----|-------|
| 客户信息完整度 | ≥95% |
| 客户跟进率 | ≥80% |
| 数据同步及时性 | ≤5分钟 |

## 2. 用户角色

### 2.1 角色定义

| 角色 | 说明 | 权限 |
|------|------|------|
| 经纪人 | 小程序端使用者，负责录入和维护客户信息 | 仅能看到自己录入的客户 |
| 区域经理 | PC端管理者，查看管辖范围内的客户数据 | 可查看所有下属客户的汇总数据 |
| 管理员 | 系统管理者，全量客户数据管理 | 可查看和导出全部客户数据 |

### 2.2 端侧分工

| 端 | 职责 |
|---|------|
| 小程序端 | 客户信息录入、编辑、查看、跟进记录 |
| PC端 | 客户列表管理、数据统计、批量操作 |

## 4. 客户信息字段

### 4.1 基本信息

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 客户姓名 | 文本 | 是 | 客户真实姓名 |
| 联系电话 | 手机号 | 是 | 11位手机号 |
| 性别 | 单选 | 否 | 男/女 |
| 意向等级 | 单选 | 是 | 高/中/低 |
| 意向楼盘 | 多选 | 否 | 关联楼盘列表 |
| 客户来源 | 单选 | 否 | 线上获客/转介绍/自然到访 |
| 备注 | 多行文本 | 否 | 其他补充信息 |

### 4.2 跟进信息

| 字段 | 类型 | 说明 |
|------|------|------|
| 跟进人 | 文本 | 经纪人姓名（自动填充） |
| 跟进时间 | 时间 | 记录时间（自动填充） |
| 跟进内容 | 多行文本 | 跟进描述 |
| 下次跟进时间 | 时间 | 计划下次跟进时间 |

## 6. 状态定义

### 6.1 客户意向等级

| 等级 | 标识颜色 | 说明 |
|------|---------|------|
| 高意向 | 红色 | 有明确购房意向，可立即跟进 |
| 中意向 | 黄色 | 有购房需求，需要持续跟进 |

| 状态 | 说明 |
|------|------|
| 跟进中 | 正在跟进 |

| 已流失 | 明确不购买或无法联系 |

## 7. 验收标准

### US-01：客户列表查看

**验收标准**：
```
Given 经纪人进入客户列表
When 查看我的客户
Then 显示该经纪人的所有客户
  - 按最新跟进时间倒序排列
  - 显示：姓名、电话、意向等级、意向楼盘
```

### US-02：新增客户

**验收标准**：
```
Given 经纪人点击新增客户
When 填写客户信息并保存
Then 客户信息保存成功
  - 姓名必填
  - 电话必填，格式校验
  - 意向等级必填
  - 保存后跳转至客户详情页
```

### US-03：编辑客户

**验收标准**：
```
Given 经纪人在客户详情页
When 点击编辑按钮
Then 进入编辑表单
  - 表单预填充当前信息
  - 修改后保存成功
  - 返回详情页显示更新后的信息
```

### US-04：删除客户

**验收标准**：
```
Given 经纪人在客户详情页
When 点击删除按钮
Then 弹出确认框
  - 确认后删除客户
  - 取消则不删除
  - 删除后跳转至客户列表
```

### US-05：PC端客户管理

**验收标准**：
```
Given 区域经理进入客户管理
When 查看客户列表
Then 显示管辖范围内的所有客户
  - 可按经纪人筛选
  - 可按楼盘筛选
  - 可按意向等级筛选
  - 可导出客户数据
```

## 8. 数据流向

```
┌─────────────┐     新增/编辑     ┌─────────────┐
│  小程序端   │ ────────────────→ │   后端API   │
│  客户管理   │ ←─────────────── │   服务端   │
└─────────────┘     返回结果      └──────┬──────┘
                                         │
                                         ↓
┌─────────────┐     查询/导出     ┌─────────────┐
│   PC端      │ ←─────────────── │   数据库    │
│  客户管理   │ ────────────────→ │            │
└─────────────┘     统计数据     └─────────────┘
```

## 9. 注意事项

1. **隐私保护**：客户电话需脱敏显示（138****8888）
2. **数据权限**：经纪人只能查看自己的客户，区域经理可查看下属客户
3. **数据同步**：小程序端和PC端数据实时同步
4. **删除限制**：已成交客户不允许删除
