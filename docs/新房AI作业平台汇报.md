# 新房AI作业平台

## 项目汇报

***

## 📋 项目概述

**新房AI作业平台**是一套面向房产经纪人的智能化作业系统，涵盖从新盘管理、探盘任务、录音分析、物料生成到客户管理的全流程功能。

### 核心价值

| 价值点      | 说明                   |
| -------- | -------------------- |
| 🔄 流程自动化 | AI赋能，自动化生成内容，提升作业效率  |
| 📊 数据可视化 | 实时数据看板，洞察业务状态        |
| 🤖 智能化工具 | AI录音分析、智能物料生成，降低人力成本 |
| 📱 移动端支持 | 小程序随时随地作业，便捷高效       |

### 用户角色

| 角色         | 说明               | 使用端       |
| ---------- | ---------------- | --------- |
| 管理员/新房维护人员 | 负责上线新盘、配置任务、管理团队 | PC端       |
| 经纪人（包含区经理） | 执行探盘任务、上传录音、AI训练 | PC端 + 小程序 |

***

## 🖥️ PC端功能模块

### 1. 新盘管理

**功能描述**：管理楼盘基础信息，包括项目详情、户型图、周边配套、销讲内容等。

**核心功能**：

| 功能   | 说明              |
| ---- | --------------- |
| 楼盘列表 | 展示所有楼盘，支持搜索、筛选  |
| 楼盘详情 | 基础信息、户型、配套、销讲内容 |
| 楼盘上线 | 将楼盘上线到探盘任务系统    |
| 回收站  | 已下线/删除的楼盘       |

**原型地址**：<https://prototypes-ivory-nine.vercel.app/pc/new-building/list.html>

***

### 2. 探盘任务

**功能描述**：管理员（我们需要wei）下发探盘任务，经纪人执行打卡、生成内容、提交报告的全流程管理。

**核心功能**：

| 功能   | 说明                |
| ---- | ----------------- |
| 任务列表 | 展示所有任务，支持状态筛选     |
| 创建任务 | 选择楼盘、配置打卡规则、设置执行人 |
| 任务分发 | 批量/单条下发任务给经纪人     |
| 任务追踪 | 实时查看任务执行进度        |
| 任务催办 | 对进行中的任务发送催办提醒     |

**任务流程**：

```
创建任务 → 配置规则 → 下发任务 → 执行打卡 → 生成内容 → 提交报告
```

**打卡配置**（与小程序一致）：

- 打卡范围、偏移阈值（200m内正常/200-500m警告/>500m拦截）
- AI自动审核开关

**原型地址**：`/pc/task/list.html`

***

### 3. 录音分析

**功能描述**：上传经纪人录音，AI自动分析话术质量，生成改进建议。

**核心功能**：

| 功能   | 说明                       |
| ---- | ------------------------ |
| 录音上传 | 支持批量上传经纪人邀约录音            |
| AI分析 | 自动分析话术规范、情绪识别、关键词覆盖、邀约意向 |
| 结果展示 | 直观展示分析结果和问题点             |
| 模板管理 | 配置不同分析维度的模板              |

**分析维度**：

- 话术规范 ✅
- 情绪识别 ✅
- 关键词覆盖 ✅
- 邀约意向 ✅

**原型地址**：`/pc/recording/list.html`

***

### 4. 物料管理

**功能描述**：统一管理探盘生成的物料（文案、配图、视频），支持审核和分发。

**核心功能**：

| 功能   | 说明          |
| ---- | ----------- |
| 物料列表 | 展示所有生成的物料   |
| 物料审核 | 审核生成的物料内容   |
| 物料分发 | 将物料分发给经纪人使用 |

**原型地址**：`/pc/material/list.html`

***

### 5. 客户管理

**功能描述**：管理潜在客户信息，跟踪客户状态，关联探盘任务。

**核心功能**：

| 功能   | 说明          |
| ---- | ----------- |
| 客户列表 | 展示所有客户，支持筛选 |
| 客户详情 | 查看客户信息和跟进记录 |
| 新增客户 | 录入新客户信息     |

**原型地址**：`/pc/customer/list.html`

***

### 6. 数据看板

**功能描述**：可视化展示各项业务数据，辅助管理层决策。

**核心数据**：

| 指标    | 说明              |
| ----- | --------------- |
| 新盘统计  | 各楼盘上线数量、状态分布    |
| 任务进度  | 任务完成率、执行中/已完成统计 |
| 物料统计  | 各类型物料生成数量       |
| 经纪人表现 | 经纪人任务完成排名       |

**原型地址**：`/pc/dashboard/overview.html`

***

## 📱 小程序端功能模块

### 1. 服务工具（首页）

**功能描述**：小程序首页入口，展示核心功能快捷入口。

**核心功能**：

| 功能     | 说明                   |
| ------ | -------------------- |
| 快捷入口   | 录音上传、内容日历等           |
| 底部胶囊导航 | 服务工具/AI设计/claw/日历/我的 |

**原型地址**：`/wxapp/home/tools.html`

***

### 2. 内容日历

**功能描述**：展示探盘任务日历视图，方便经纪人安排工作。

**核心功能**：

| 功能   | 说明              |
| ---- | --------------- |
| 日历视图 | 按日期展示任务         |
| 任务卡片 | 显示任务状态、类型       |
| 任务下发 | 点击下发任务卡片，跳转到配置页 |

**任务状态**（小程序仅显示待下发）：

- 待下发 → 跳转配置页进行下发

**原型地址**：`/wxapp/home/calendar.html`

***

### 3. 任务配置

**功能描述**：配置探盘任务各项参数，与PC端保持一致。

**配置项**（与PC一致）：

| 配置分组   | 说明             |
| ------ | -------------- |
| 楼盘信息   | 只读，展示楼盘基础信息    |
| 打卡配置   | 打卡范围、偏移阈值      |
| 采集点位   | 点位列表（不可删除/可删除） |
| 内容生成   | 文案生成、图片/视频生成开关 |
| 话术训练   | 启用开关、训练项目      |
| 录音分析   | 启用开关、分析维度      |
| 任务截止时间 | 任务结束时间         |
| 提醒配置   | 轻度/中度/重度提醒     |
| 执行人员   | 选择执行经纪人        |
| 通知人员   | 任务通知给谁         |

**原型地址**：`/wxapp/task/checkin-config.html`

***

### 4. 任务执行

**功能描述**：经纪人执行探盘任务的完整流程。

**执行流程**：

```
定位打卡 → 点位采集 → 内容生成 → 文案确认
    ↓          ↓          ↓          ↓
  拍照定位   拍照/视频   AI生成     选择发布
            填写内容   配图视频
```

**页面说明**：

| 页面                    | 功能                    |
| --------------------- | --------------------- |
| location-checkin.html | 阶段1：定位校验，确认已到达楼盘      |
| point-checkin.html    | 阶段2：点位资料收集，拍照/视频/文字   |
| content-gen.html      | 阶段3：内容生成，AI生成文案/配图/视频 |
| confirm-script.html   | 阶段4：文案确认，选择发布的物料      |

**返回路径**：

```
配置页 → 任务详情 → 定位打卡 → 点位打卡 → 内容生成 → 文案确认
                    ↓          ↓           ↓           ↓
                  日历      location   point      content
```

**原型地址**：`/wxapp/task/detail.html`

***
