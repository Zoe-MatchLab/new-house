## 需要补充的交互状态

### 1. 全局状态

| 状态 | 说明 | 实现方式 |
|-----|------|---------|
| 加载中 | 数据获取中 | 骨架屏/Spinner |
| 空数据 | 无数据 | 空状态插图+文案 |
| 错误 | 网络异常/服务异常 | 错误页+重试按钮 |
| 成功 | 操作成功 | Toast提示 |

### 2. 表单状态

| 状态 | 说明 | 实现方式 |
|-----|------|---------|
| 默认 | 正常可输入 | - |
| 聚焦 | 正在输入 | 蓝色边框 |
| 错误 | 校验失败 | 红色边框+错误文案 |
| 禁用 | 不可操作 | 灰色背景 |

### 3. 按钮状态

| 状态 | 说明 | 实现方式 |
|-----|------|---------|
| 默认 | 可点击 | 主色背景 |
| Hover | 鼠标悬停 | 颜色加深10% |
| 点击中 | 正在提交 | 显示loading+禁用 |
| 禁用 | 不可点击 | 灰色背景 |

### 4. 图片上传状态

| 状态 | 说明 | 实现方式 |
|-----|------|---------|
| 待上传 | 可拖拽上传 | 虚线边框 |
| 上传中 | 正在上传 | 进度条+百分比 |
| 上传成功 | 已上传 | 缩略图+删除按钮 |
| 上传失败 | 上传出错 | 错误图标+重试按钮 |

### 5. 分页状态

| 状态 | 说明 | 实现方式 |
|-----|------|---------|
| 默认 | 正常展示 | - |
| 加载中 | 正在请求 | 表格区域显示loading |
| 跳转中 | 正在翻页 | 分页控件禁用 |
