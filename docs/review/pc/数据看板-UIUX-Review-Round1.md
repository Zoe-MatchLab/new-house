## 二、视觉设计问题

### 2.1 颜色问题

| 问题 | 位置 | 建议 |
|-----|------|-----|
| 主色使用过多 | 多个卡片都用primary蓝色 | 按指标类型分配颜色（成功=绿色、警告=橙色、危险=红色） |
| 渐变色过于花哨 | 进度条渐变 | 使用纯色或简单双色渐变 |
| 漏斗图颜色递减不合理 | 漏斗从上到下颜色变浅 | 应保持语义一致或用色阶 |

**建议色板**：
```
Primary: #165DFF (主色)
Success: #00B42A (成功/正向)
Warning: #FF7D00 (警告/注意)
Danger: #F53F3F (危险/负向)
Purple: #722ED1 (辅助色)
Gray: #86909C (次要文字)
```

### 2.2 字体问题

| 问题 | 位置 | 建议 |
|-----|------|-----|
| 指标数值过大 | metric-value: 28px | 重要数据可增至32-36px |
| 标签文字过小 | metric-label: 13px | 建议14px保证可读性 |
| 趋势数字颜色对比度不足 | metric-trend | 白色文字在透明背景上可读性差 |

### 2.3 间距问题

| 问题 | 位置 | 建议 |
|-----|------|-----|
| 卡片内边距不统一 | 部分12px，部分20px | 统一为16px或20px |
| 组件间距过紧 | 指标卡片间距 | gap调整为20-24px |
| Tab与内容区距离 | 16px | 增至24px增加呼吸感 |

## 三、布局问题

### 3.1 网格系统

**当前问题**：
- 4列网格在部分分辨率下显示拥挤
- 固定宽度可能导致内容溢出

**建议**：
```
桌面(>1440px): 4列
笔记本(1280-1440px): 3列
平板(1024-1280px): 2列
```

### 3.2 漏斗图布局

**当前问题**：
- 漏斗图与旁边数据卡片高度不对齐
- 漏斗箭头位置不精确

**建议**：
```
┌─────────────────────────┬──────────────┐
│      漏斗图区域          │   转化率    │
│                         │   数据卡片   │
│   [报备数 3,856]        │             │
│        ↓ 33.3%          │   33.3%    │
│   [带看组数 1,286]      │   9.9%     │
│        ↓ 9.9%            │   22.5%    │
│   [成交单量 128]        │             │
└─────────────────────────┴──────────────┘
```

## 四、数据可视化问题

### 4.1 图表类型

| 现有图表 | 问题 | 建议 |
|---------|------|-----|
| 折线图 | 缺少数据点标记 | 添加pointRadius |
| 柱状图 | 颜色单一 | 按系列使用不同颜色 |
| 饼图 | 标签重叠风险 | 使用图例或调整布局 |
| 漏斗图 | 样式较简单 | 增加hover效果和动画 |

### 4.2 图表交互

| 缺失交互 | 建议 |
|---------|-----|
| Hover效果 | 显示具体数值tooltip |
| 点击下钻 | 点击数据查看明细 |
| 图例交互 | 点击图例显示/隐藏系列 |
| 缩放 | 时间范围可拖拽选择 |

### 4.3 雷达图（五维能力）

**当前问题**：
- 尺寸过小，标签重叠
- 缺少参考线（如行业平均）

**建议**：
```
- 增大canvas尺寸至350px
- 缩短标签文字
- 添加参考圆环（60分、80分基准线）
- hover显示各维度详细分数
```

## 五、交互体验问题

### 5.1 按钮状态

| 状态 | 当前实现 | 建议 |
|-----|---------|-----|
| Default | 仅有边框 | 添加hover背景色 |
| Hover | 无变化 | 背景色加深10% |
| Active | 无变化 | 缩小transform: scale(0.98) |
| Disabled | 灰色文字 | 添加disabled样式 |
| Loading | 无 | 添加spinner图标 |

### 5.2 卡片交互

| 问题 | 建议 |
|-----|-----|
| 不可点击的卡片缺少cursor | 鼠标指针提示 |
| 点击区域过小 | 扩大可点击区域 |
| 缺少hover提升效果 | 添加shadow或border高亮 |

### 5.3 筛选器交互

| 问题 | 建议 |
|-----|------|
| 下拉框打开无动画 | 添加transition |
| 选中状态不明显 | 添加选中样式 |
| 多选筛选项 | 显示已选数量badge |

### 5.4 Tab切换

| 问题 | 建议 |
|-----|------|
| 切换无过渡动画 | 添加fade或slide动画 |
| 当前Tab下划线不明显 | 加粗或添加背景色 |
| 内容区加载无反馈 | 添加skeleton或spinner |

## 七、可访问性问题

### 7.1 颜色对比度

| 位置 | 当前对比度 | WCAG标准 | 建议 |
|-----|----------|---------|-----|
| 白色文字在透明背景 | ~3:1 | 4.5:1 | 添加半透明黑色背景 |
| 灰色次要文字 | ~2.5:1 | 3:1 | 加深至#4E5969 |
| 图表标签 | ~2:1 | 3:1 | 添加背景色或加粗 |

### 7.2 键盘导航

| 问题 | 建议 |
|-----|------|
| Tab筛选器无焦点样式 | 添加focus:ring |
| 下拉框无法键盘操作 | 添加keyboard支持 |
| 图表无法键盘访问 | 添加aria-label |

### 7.3 屏幕阅读器

| 问题 | 建议 |
|-----|------|
| 图表缺少alt文本 | 添加aria-label描述 |
| 动态数据无通知 | 使用aria-live区域 |
| 图例缺少说明 | 添加sr-only文本 |

## 八、性能问题

### 8.1 渲染性能

| 问题 | 建议 |
|-----|------|
| Chart.js实例未销毁 | 页面切换时destroy |
| 大量数据未分页 | 使用虚拟滚动 |
| 图片未压缩 | 使用WebP格式 |

### 8.2 首屏加载

| 问题 | 建议 |
|-----|------|
| CDN资源阻塞 | 添加preconnect |
| Chart.js较大 | 按需加载图表类型 |
| 字体加载闪烁 | 使用font-display: swap |

## 九、改进优先级

### P0（必须修复）

1. **漏斗图样式优化** - 颜色、间距、动画
2. **按钮状态完善** - hover、active、disabled、loading
3. **加载/空/错误状态** - 添加skeleton、空状态插画
4. **颜色对比度修复** - 达到WCAG AA标准

### P1（应该修复）

5. **Tab切换动画** - 添加过渡效果
6. **图表交互增强** - tooltip、点击下钻
7. **筛选器交互优化** - 动画、选中状态
8. **雷达图尺寸调整** - 增大并优化标签

### P2（可以修复）

9. **键盘导航支持** - Tab焦点、键盘操作
10. **屏幕阅读器支持** - aria-label、aria-live
11. **图表导出功能** - PNG/SVG导出
12. **响应式布局调整** - 适配不同屏幕宽度
