## 二、数据看板功能单元

### 2.1 PC管理后台看板

#### 单元A：内容发布指标

| 指标 | 说明 | 数据来源 |
|-----|------|---------|
| 发布量 | 各平台发布数量 | 物料管理 |
| 浏览量 | 内容曝光量 | 渠道统计 |
| 投流费用 | 付费推广消耗 | 广告平台 |

#### 单元B：获客转化指标

| 指标 | 说明 | 数据来源 |
|-----|------|---------|
| 咨询量 | 客户主动咨询次数 | 小程序 |
| 线索量 | 留资客户数量 | 小程序 |
| 报备量 | 客户报备楼盘数量 | 探盘任务 |
| 带看量 | 经纪人带看次数 | 探盘任务 |
| 获客成本 | 单个有效客户成本 | 计算得出 |

#### 单元C：产品力训练指标

| 指标 | 说明 | 数据来源 |
|-----|------|---------|
| 专家占比 | 各级别专家人数占比 | 话术训练 |
| 平均能力分 | 团队整体能力均值 | 话术训练 |
| 能力提升率 | 较上一周期能力提升幅度 | 话术训练 |

#### 单元D：任务执行指标

| 指标 | 说明 | 数据来源 |
|-----|------|---------|
| 任务完成率 | 已完成/总任务数 | 探盘任务 |
| 任务逾期率 | 逾期任务/总任务数 | 探盘任务 |
| 人均任务数 | 任务总数/经纪人数量 | 探盘任务 |

#### 单元E：楼盘分析指标

| 指标 | 说明 | 数据来源 |
|-----|------|---------|
| 楼盘覆盖率 | 已上线楼盘/总楼盘 | 新盘管理 |
| 物料丰富度 | 平均物料数量/楼盘 | 物料管理 |
| 热门楼盘 | 按咨询量/带看量排名 | 综合 |

## 四、需要讨论的问题

### 问题1：数据来源确认

| 数据类型 | 获取方式 | 说明 |
|---------|---------|------|
| 公众号浏览量 | ✅ API获取 | 可通过公众号后台API |
| 公众号投流费用 | ✅ API获取 | 可通过广告后台API |
| 小红书数据 | ⚠️ 数据导入 | 手动导入CSV/Excel |
| 视频号数据 | ⚠️ 数据导入 | 手动导入CSV/Excel |
| 朋友圈数据 | ⚠️ 数据导入 | 手动导入CSV/Excel |
| 咨询量/线索量 | ✅ 小程序统计 | 小程序内埋点 |
| 带看量 | ✅ 小程序确认 | 经纪人提交带看记录 |

### 问题2：数据更新频率

| 选项 | 结果 |
|-----|------|
| **用户主动刷新** | ✅ 已确认 |

说明：数据不会自动更新，用户点击刷新按钮后获取最新数据

### 问题3：数据权限

| 项目 | 结果 |
|-----|------|
| 端 | PC端管理后台 |
| 权限 | 管理员权限（全部数据） |
| 更新方式 | 用户主动刷新 |
