## 二、数据看板功能单元

### 2.1 PC管理后台看板

#### 单元A：内容发布指标

| 指标 | 说明 | 数据来源 |
|-----|------|---------|
| 发布量 | 各平台发布数量 | 物料管理 |
| 浏览量 | 内容曝光量 | 渠道统计 |
| 投流费用 | 付费推广消耗 | 广告平台 |

#### 单元B：获客转化指标

| 指标 | 说明 | 数据来源 |
|-----|------|---------|
| 咨询量 | 客户主动咨询次数 | 小程序 |
| 线索量 | 留资客户数量 | 小程序 |
| 报备量 | 客户报备楼盘数量 | 探盘任务 |
| 带看量 | 经纪人带看次数 | 探盘任务 |
| 获客成本 | 单个有效客户成本 | 计算得出 |

#### 单元C：产品力训练指标

| 指标 | 说明 | 数据来源 |
|-----|------|---------|
| 专家占比 | 各级别专家人数占比 | 话术训练 |
| 平均能力分 | 团队整体能力均值 | 话术训练 |
| 能力提升率 | 较上一周期能力提升幅度 | 话术训练 |

#### 单元D：任务执行指标

| 指标 | 说明 | 数据来源 |
|-----|------|---------|
| 任务完成率 | 已完成/总任务数 | 探盘任务 |
| 任务逾期率 | 逾期任务/总任务数 | 探盘任务 |
| 人均任务数 | 任务总数/经纪人数量 | 探盘任务 |

#### 单元E：楼盘分析指标

| 指标 | 说明 | 数据来源 |
|-----|------|---------|
| 楼盘覆盖率 | 已上线楼盘/总楼盘 | 新盘管理 |
| 物料丰富度 | 平均物料数量/楼盘 | 物料管理 |
| 热门楼盘 | 按咨询量/带看量排名 | 综合 |

## 四、需要讨论的问题

### 问题1：数据来源确认

以下数据来源是否能获取？

| 数据类型 | 问题 |
|---------|-----|
| 浏览量/曝光量 | 是否需要接入各平台API？ |
| 投流费用 | 是否需要接入广告平台？ |
| 咨询量/线索量 | 小程序能否统计？ |
| 带看量 | 如何确认带看？ |

### 问题2：数据更新频率

| 选项 | 说明 |
|-----|------|
| 实时 | 操作后即时更新 |
| 小时级 | 每小时刷新 |
| 日更 | 每日凌晨更新 |

### 问题3：数据权限

是否需要按门店/区域隔离数据？

| 选项 | 说明 |
|-----|------|
| 是 | 各门店只看本店数据 |
| 否 | 区域经理以上可看全部 |
