## 一、系统概述

### 1.1 产品定位

新房AI作业平台是一款面向房地产经纪人的智能作业工具，通过AI能力提升经纪人专业能力和作业效率，同时为管理层提供数据支撑。

### 1.2 目标用户

| 用户角色 | 说明 | 使用端 |
|---------|------|-------|
| 新房维护人员 | 负责录入和管理楼盘信息 | PC |
| 区域经理 | 负责任务下发、团队管理、数据查看 | PC + 小程序 |
| 经纪人 | 执行探盘任务、使用AI工具、管理客户 | 小程序为主 |
| 审核人员 | 审核楼盘信息、物料内容 | PC |

### 1.3 核心价值

| 价值点 | 说明 |
|-------|------|
| 快速了解楼盘 | 结构化楼盘信息，图文并茂 |
| AI话术训练 | 智能对话练习，实时反馈 |
| 自动生成内容 | AI生成文案配图，降低成本 |
| 数据驱动管理 | 实时看板，掌握团队状态 |

## 三、详细功能需求

### 3.1 新盘管理模块（PC端）

#### 3.1.1 功能概述

| 功能 | 说明 | 优先级 |
|-----|------|-------|
| 楼盘列表 | 查看所有楼盘，支持搜索筛选 | P0 |
| 新增楼盘 | 录入楼盘完整信息（81字段） | P0 |
| 编辑楼盘 | 修改楼盘信息 | P0 |
| 删除楼盘 | 软删除至回收站 | P1 |
| 回收站 | 恢复/永久删除楼盘 | P1 |
| 上线配置 | 设置楼盘上线/下线 | P0 |
| 楼盘详情 | 查看楼盘完整信息 | P0 |

#### 3.1.2 楼盘信息字段（81字段）

| 分类 | 字段数 | 说明 |
|-----|-------|------|
| 基本信息 | 15 | 名称、地址、开发商、均价等 |
| 产品信息 | 20 | 户型、面积、楼层、装修等 |
| 配套信息 | 18 | 地铁、商场、医院、学校等 |
| 销讲信息 | 15 | 核心卖点、话术、FAQ等 |
| 销售信息 | 8 | 销控表、优惠政策、预约方式等 |
| 媒体信息 | 5 | 封面图、户型图、样板间图等 |

#### 3.1.3 页面清单

| 页面 | 路径 | 功能 |
|-----|------|-----|
| 新盘列表 | /pc/new-building/list.html | 楼盘搜索、筛选、新增入口 |
| 新增楼盘 | /pc/new-building/edit.html | 表单录入、图片上传 |
| 编辑楼盘 | /pc/new-building/edit.html?id=xxx | 表单编辑 |
| 楼盘详情 | /pc/new-building/detail.html?id=xxx | 完整信息展示 |
| 上线配置 | /pc/new-building/publish.html?id=xxx | 上下线管理 |
| 回收站 | /pc/new-building/recycle.html | 恢复/删除 |

#### 3.1.4 业务流程

```
新增楼盘流程：
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│ 填写信息 │ → │ 上传图片 │ → │ 保存草稿 │ → │ 提交审核 │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
                                                     ↓
                                            ┌─────────┐
                                            │ 审核通过 │
                                            │ 上线发布 │
                                            └─────────┘
```

#### 3.1.5 验收标准

| 功能点 | 验收条件（Given-When-Then） | 优先级 |
|-------|---------------------------|-------|
| 新增楼盘 | Given 用户进入新增楼盘页，When 填写必填字段并上传图片，Then 保存成功 | P0 |
| 表单验证 | Given 用户未填写必填字段，When 点击保存，Then 提示必填项 | P0 |
| 图片上传 | Given 用户选择图片，When 图片≤10MB，Then 上传成功并展示 | P0 |
| 搜索楼盘 | Given 用户在列表页，When 输入楼盘名称，Then 显示匹配的楼盘 | P0 |
| 状态筛选 | Given 用户在列表页，When 选择状态筛选，Then 显示该状态楼盘 | P0 |

### 3.2 探盘任务模块（PC + 小程序）

#### 3.2.1 功能概述

| 功能 | PC端 | 小程序端 | 优先级 |
|-----|------|---------|-------|
| 任务列表 | ✅ | ✅ | P0 |
| 创建任务 | ✅ | ❌ | P0 |
| 任务详情 | ✅ | ✅ | P0 |
| 任务配置 | ✅ | ✅ | P0 |
| 定位打卡 | ❌ | ✅ | P0 |
| 点位采集 | ❌ | ✅ | P0 |
| 内容生成 | ❌ | ✅ | P0 |
| 文案确认 | ❌ | ✅ | P0 |
| 任务追踪 | ✅ | ❌ | P0 |
| 进度看板 | ✅ | ❌ | P1 |

#### 3.2.2 PC端功能详述

**任务列表**
- 任务名称、楼盘、状态、执行人、截止时间
- 支持按状态筛选（待下发/待执行/执行中/已完成）
- 支持按楼盘、执行人筛选
- 支持批量操作（下发、删除）

**创建任务**
- 选择楼盘（从已上线楼盘中选择）
- 配置打卡规则：
  - 打卡范围（米）
  - 位置偏移阈值（米）
  - AI自动审核开关
- 配置点位（需要采集的POI点）
- 配置内容生成规则
- 配置话术训练要求
- 配置执行人员
- 设置截止时间
- 设置可见范围

**任务追踪**
- 甘特图展示任务进度
- 执行人员完成状态
- 催办功能

#### 3.2.3 小程序端功能详述

**内容日历**
- 日历视图展示任务
- 任务卡片显示楼盘、状态
- 点击进入任务详情
- 下发入口

**配置任务（PC下发后的执行配置）**
- 查看打卡配置
- 查看点位配置
- 查看内容生成配置
- 查看话术训练配置
- 查看执行人员
- **执行下发**：点击后正式进入执行阶段

**任务详情**
- 任务说明
- 执行流程展示
- 当前进度

**定位打卡**
- GPS定位
- 位置偏差校验
- 打卡成功/失败提示

**点位打卡**
- 拍照采集
- 视频录制
- 文字说明
- POI信息录入

**内容生成**
- AI生成文案
- AI生成配图
- 支持编辑
- 一键发布到平台

**文案确认**
- 查看生成内容
- 修改编辑
- 确认发布
